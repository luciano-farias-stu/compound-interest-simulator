# Simulador de Crecimiento de Inversión

Simulador interactivo que compara tres estrategias:

• Ahorro sin interés  
• Fondo bancario  
• Inversión en mercado accionario  

El simulador muestra la evolución del capital en el tiempo utilizando interés compuesto.

## Características

- Formateo automático de moneda CLP
- Gráfico interactivo con Chart.js
- Animación de contadores de capital
- Banda de volatilidad del mercado
- Comparación visual entre estrategias

## Tecnologías

HTML  
CSS  
JavaScript  
Chart.js
